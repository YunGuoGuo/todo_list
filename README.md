# todo-list

## 介绍

​		基于vue3 + vuex + element3实现的简易todo-list🧑🏻‍💻 用于学习

##  使用方法

### 安装依赖

```shell
npm install
```

### 启动服务

```shell
npm run serve
# 或者
yarn serve
```
